HTML5简述

发表于 2015-10-23

HTML5

HTML5新语义标签

header -> nav

main -> aside article section

footer

其实就是一个有名字的div.由于兼容性问题,实际开发的时候 div+class 还是主流, 这些新语义标签的兼容性问题可以用html5-shiv.js来解决.

新type属性

color data number month search email tel
在这些新的type属性中, 有一部分在桌面端是无法看到效果的, 而且移动端会弹出不同的虚拟键盘 很明显的这些属性不同浏览器的兼容性也是不一样的, 我们的谷歌支持的最好, ie火狐的差别很大, 不过就算不支持, 也会显示一个普通的文本框 但这种状况只是局限于pc端, 所以在pc端使用频率并不频繁, 移动端对此支持较好.

新表单属性

autocomplete -> 自动完成, name属性必须设置, 切记所在的表单必须点击submit按钮之后才能够提示

mutilate -> 多选 一般配合 tpye=file来使用

antofocus -> 获取焦点 一般一个页面只设置一个

placeholder -> 文本框的提示信息

多媒体标签

video -> width heigth 设置的是标签的大小, 视频本身不会跟随缩放

audio -> 音乐播放器

共同的属性: src autoplay loop controls 通过source 指定多个文件, 而且这些标签所展示的音乐或者视频可以直接被右键另存为.
